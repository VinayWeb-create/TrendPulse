Every tool, explained

A complete breakdown of every metric, score, and analysis technique powering TrendPulse.

📊

Engagement Score

The Engagement Score is TrendPulse's core metric. It weighs the ratio of active interactions (likes + comments) against total views, then normalizes against the category average. A score above 70 indicates exceptional audience resonance that triggers YouTube's recommendation algorithm.

Formula
score = ((likes + comments) / views) × 100
normalized = (score / category_avg) × 100
final = min(normalized × weight_factor, 100)
🚀

Trend Potential

Trend Potential predicts how likely a video is to appear on YouTube's Trending page. It combines view velocity (speed of accumulation), like ratio, comment-to-view ratio, and how the video performs relative to the channel's historical average.

Formula
velocity = views / hours_since_publish
potential = (velocity_score × 0.40) +
  (like_ratio_score × 0.35) +
  (channel_perf_ratio × 0.25)
⚡

View Velocity

View Velocity measures how quickly a video accumulates views relative to its publish age. A video with 1M views after 2 hours has vastly different signal than one with the same count after a month. We use exponential time decay weighting.

Formula
age_hours = (now - publish_date) / 3600
raw_velocity = views / age_hours
decay = e^(-0.05 × age_hours)
velocity_score = raw_velocity × decay
👥

Subscriber Ratio

The Subscriber Ratio reveals whether a video reaches beyond the existing audience. A 100K channel getting 5M views has a 50× ratio — indicating viral breakout. This is a key signal YouTube uses to surface content to non-subscribers.

Formula
ratio = views / subscriber_count
above 1.0 → exceeds subscriber base
above 5.0 → strong viral signal
above 10.0 → exceptional breakout
💬

Comment Analysis

Comments-per-view is a strong signal of passionate engagement. YouTube's algorithm treats high comment counts as an indication of content that provokes conversation and extends watch sessions across sessions.

Signal Weights
comment_rate = comments / views × 100
above 0.1% → above average
above 0.5% → high engagement
above 1.0% → exceptional discussion
🏷️

Tag & Category Intel

Video tags and category placement directly affect discoverability. We extract all metadata tags and cross-reference against trending search queries. Category ranking shows where the video sits within its vertical.

Data Points
- All metadata tags extracted
- Category ID → human-readable name
- Category benchmark scores
- Tag relevance estimation
